Interpretación rápida

    Base teórica (Hamilton 1989 simplificado)

    Suponemos un proceso oculto Sₜ ∈ {0,1} con transición de Markov de primer orden: P(Sₜ=j | Sₜ₋₁=i)=pᵢⱼ. Dado el estado, la observación sigue una normal con parámetros específicos: yₜ | Sₜ=s ~ N(μₛ, σ²ₛ).

    El algoritmo forward-backward entrega probabilidades filtradas/suavizadas y EM maximiza la log-verosimilitud para estimar μ₀, μ₁, σ₀², σ₁², P. En esta demo, además de distinta media permitimos distinta varianza para capturar volatilidad asimétrica entre regímenes.

    Fuente de datos: API CSV pública de FRED (Federal Reserve Bank of St. Louis).
